# Student Performance Prediction

A Machine Learning project that predicts student marks based on study hours using Linear Regression.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Results
- Predicted Marks for 8 study hours: 70
- R² Score: 0.975

## Workflow
1. Data Creation
2. Data Cleaning
3. Data Visualization
4. Model Training
5. Prediction
6. Evaluation
